# Plan: UX 개선 — 내 위치 복귀 / 최근 검색 / 즐겨찾기

## 개요

ClimateGo 앱의 반복 사용성을 높이는 3가지 UX 기능을 추가한다.

## 기능 범위

### UX-01. 내 위치로 돌아가기 버튼
- 지도를 팬/줌하다가 내 위치를 잃었을 때 버튼 한 번으로 복귀
- 지도 우하단 고정 FAB 버튼
- 클릭 시 현재 GPS 위치로 지도 중심 이동 + 줌 레벨 복원

### UX-02. 최근 검색 기록
- 경로 탐색 목적지 입력창에 최근 검색 장소 표시
- localStorage 저장, 최대 5개 유지
- 검색창 포커스 시 드롭다운으로 표시
- 항목 클릭 시 바로 검색 실행

### UX-03. 즐겨찾기 정류장
- 정류장 도착 정보 패널에 즐겨찾기 토글 버튼 추가
- localStorage 저장, 최대 10개
- 앱 하단 또는 기후동행 패널 영역에 즐겨찾기 목록 표시
- 즐겨찾기 정류장 클릭 시 도착 정보 바로 조회

## 기술 접근

- 모든 데이터는 **localStorage** 저장 (서버 불필요)
- UX-01: MapView에 `map.setCenter()` 호출 — ref 통해 App에서 제어
- UX-02: HeaderSearch 컴포넌트 내 최근 검색 드롭다운
- UX-03: ArrivalPanel 즐겨찾기 버튼 + ClimateRoutesPanel 대신 표시 가능한 즐겨찾기 패널

## 비기능 요구사항

- 신규 라이브러리 없음 (React + localStorage만 사용)
- 기존 컴포넌트 수정 최소화
- 모바일 터치 친화적 버튼 크기 (최소 44px)
