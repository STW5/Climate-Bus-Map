# Plan: 기후동행카드 최적 동선 탐색

> 기후동행카드로 이용 가능한 대중교통(버스+지하철)만을 활용한 최적 경로 탐색 기능

- **작성일**: 2026-03-09
- **레벨**: Dynamic
- **상태**: 대기 중 (Phase 4 API 안정화 이후 시작)

---

## 1. 문제 정의

### 배경

현재 앱은 "내 주변에 어떤 기후동행 버스가 있는가"만 알 수 있다.
하지만 실제 사용자는 **목적지까지 기후동행카드로 어떻게 가는지**를 알고 싶다.

기후동행카드는 서울 시내버스 + 지하철(1~9호선 등) + 따릉이를 포괄하는 카드이므로,
버스만이 아닌 지하철을 포함한 경로 탐색이 필요하다.

### 핵심 문제

- 목적지까지 경로는 T-Map/카카오 등에서 나오지만 기후동행카드 사용 가능 여부가 표시되지 않음
- 환승 경로 중 일부 구간이 기후동행카드 불가인 경우 사용자가 인지하기 어려움
- 지하철은 현재 DB에 기후동행 가능 노선 데이터가 없음

### 해결 목표

> 출발지→목적지 경로 중 기후동행카드로 100% 이용 가능한 경로만 추천

---

## 2. 핵심 기능

### F-01: 지하철 기후동행 가능 노선 DB 추가

- 기후동행카드 사용 가능한 지하철 노선/구간 데이터를 DB에 추가
- 버스와 동일하게 `climate_eligible_routes` 테이블 또는 별도 테이블로 관리
- 서울시 공식 기후동행카드 홈페이지 기준으로 데이터 구성

### F-02: T-Map 대중교통 경로 API 연동

- T-Map 대중교통 경로 탐색 API 사용 (이미 SDK 연동되어 있음)
- 출발지/목적지 입력 UI 추가
- 반환된 경로의 각 구간(버스 노선 ID, 지하철 노선)을 DB와 대조

### F-03: 기후동행 가능 경로 필터링

- 경로의 모든 구간이 기후동행카드 사용 가능한 경우만 표시
- 일부 구간만 가능한 경우: 표시하되 불가 구간을 시각적으로 구분
- 기후동행 100% 가능 경로 우선 정렬

### F-04: 경로 결과 UI

- 출발~도착 경로 카드 형태 표시
- 각 구간별 기후동행 가능 여부 배지 표시
- 기후동행 불가 구간이 있을 경우 경고 안내

## 3. 기술 검토

### 경로 탐색: ODsay API + T-Map SDK 조합 (확정 2026-03-09)

T-Map 대중교통 경로 API는 무료 10건/일로 사실상 유료(건당 0.88원)이므로,
**ODsay API(무료 1,000건/일)로 경로 데이터를 받고 T-Map 지도에 렌더링**하는 방식으로 확정.

- ODsay API 키: 발급 완료, `VITE_ODSAY_API_KEY`로 FE `.env`에 등록
- 엔드포인트: `GET https://api.odsay.com/v1/api/searchPubTransPathT`
- 응답 구조:
  ```
  result.path[]
    └── info.trafficType   1=지하철, 2=버스, 3=도보
    └── subPath[]
          ├── trafficType  1=지하철, 2=버스, 3=도보
          ├── lane[].busNo       버스 번호
          ├── lane[].busID       버스 노선 ID ← climate_eligible_routes.route_id와 대조
          ├── lane[].subwayCode  지하철 호선 코드 ← 화이트리스트와 대조
          └── passStopList.stations[].x, y  ← T-Map 폴리라인용 좌표
  ```
- **버스**: `lane[].busID` → DB `climate_eligible_routes.route_id` 직접 대조 가능 ✅
- **지하철**: `lane[].subwayCode` → 호선 화이트리스트(1~9호선, 신분당선 제외)로 판단 ✅
- 경로 좌표는 T-Map SDK `Tmapv2.Polyline()`으로 지도에 표시

### 지하철 기후동행 범위 (하드코딩으로 관리)

별도 API 없음. 서울시 공식 고시 기준으로 상수 관리.

- **적용**: 1~9호선, 경의중앙선, 수인분당선, 경춘선, 우이신설선, 신림선, 공항철도
- **제외**: 신분당선, GTX
- **예외 구간**: 5호선(미사~하남검단산), 7호선(석남~까치울), 공항철도(인천공항 1~2터미널) 하차 가능
- 변경 빈도가 낮아 하드코딩이 DB보다 적합
