# Plan: 경로탐색 출발지 선택 기능

> 경로탐색 출발지를 내 위치 고정에서 자유롭게 선택 가능하도록 개선

- **작성일**: 2026-03-17
- **레벨**: Dynamic
- **상태**: 계획 중

---

## 1. 문제 정의

현재 경로탐색은 출발지가 `position`(내 현재 위치)으로 고정되어 있어
다른 위치에서 출발하는 경로를 탐색할 수 없다.

---

## 2. 해결 방법

`RouteSearchPanel`에 출발지 입력 필드 추가:
- 기본값: 내 위치 (현재 GPS 좌표)
- 텍스트 입력 시: T-Map POI 검색으로 좌표 변환 (목적지와 동일한 방식)
- 내 위치 버튼: 출발지를 현재 위치로 초기화

---

## 3. 구현 계획

1. `RouteSearchPanel.jsx` — 출발지 입력 UI 추가
   - 출발지 input (기본: "내 위치")
   - 내 위치 버튼 (GPS 아이콘)
   - 목적지 input (기존)
2. `App.jsx` — `handleRouteSearch` 수정
   - `start` 파라미터를 `position` 고정 → 선택된 출발지로 변경
3. `tmapApi.js` — 출발지 POI 검색 (목적지 검색과 동일 로직, 재사용)

---

## 4. 범위

- FE 전용, BE 변경 없음
- 출발지 미입력 시 현재 위치 폴백
